# Opencart Docker-Compose

Clone this repo 
```
git clone https://github.com/als698/opencart-dc.git .
```
and run
```
docker-compose up -d
```

## Config
Opencart: http://localhost/  
Opencart Admin: http://localhost/admin/  
phpMyAdmin: http://localhost:81/  
Database port: 3306
